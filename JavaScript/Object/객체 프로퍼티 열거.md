## 객체 프로퍼티 열거하기

```js
const obj = {
	name: 'suin',
	age: 18,
	address: 'seoul',
};
```

---

객체의 모든 프로퍼티를 열거하려면 for ~ in 문을 사용하자.
<br/>

문법

```js
for(변수 선언문 prop in 객체 ) {}
```

```js
//프로퍼티만 열거
for (const key in obj) {
	console.log(key); //"name" "age" "address"
}

//프로퍼티와 값 둘 다 열거
for (const key in user) {
	console.log(user[key]);
	// 'name' 'suin'
	// 'age' 18
	// 'address' 'seoul'
}

// 값만 열거
for (const key in user) {
	console.log(user[key]);
	//'suin'
	//18
	//'seoul'
}
```
